## Touch Targets

All interactive elements meet WCAG 2.1 AA standards:
- Accordion buttons: 68px min height
- Search input: 56px height
- Clear button: 44px x 44px
- CTA buttons: 48px height
- Chevron icons: 44px x 44px clickable area

## Responsive Behavior

### Mobile (< 640px)
- Single column layout
- Full width elements
- Stacked buttons in footer
- Touch-friendly 44px targets
- Larger padding for thumb access

### Tablet (640px - 1024px)
- Slightly larger text
- More padding/spacing
- Grid layout for footer buttons
- Optimized search bar width

### Desktop (> 1024px)
- Max-width container (896px)
- Centered content
- Inline footer buttons
- Hover states more prominent
- Larger text sizes
